Na área de interfaces com o usuário, um ambiente de desktop é um sistema que oferece soluções com interface gráfica com o usuário (GUI) para operar um computador. O nome é derivado da metáfora de desktop, ou "tampo de mesa", hoje usado por essas interfaces. Um ambiente de desktop fornece ícones, pastas, barra de ferramentas, papéis de parede e habilidades como arrastar e soltar.Um ambiente de desktop muito comum em computadores pessoais é aquele provido pelo Microsoft Windows; outro ambiente comum é aquele provido pelo Mac OS X.Importantes ambientes de desktop para sistemas Unix-like que utilizam o X Window System (como o sistema GNU/Linux) includem KDE, GNOME, Xfce e CDE.

Em estatística e econometria, ARIMA é o nome dado a um modelo muito utilizado na modelagem e previsões séries temporais. O termo deriva do inglês autoregressive integrated moving average, que significa modelo auto-regressivo integrado de média móvel. O modelo foi sistematizado em 1976 pelos estatísticos George Box e Gwilym Jenkins, o que torna o modelo conhecido também por Modelo de Box-Jenkins.O modelo ARIMA é uma generalização do modelo auto-regressivo de média móvel (ARMA). A representação ARIMA(p, d, q) refere-se, respectivamente, às ordens de auto-regressão, de integração e de média móvel:No caso de d = 0, temos o modelo ARMA(p, q), e no caso de também q = 0, temos os modelo AR(p). O modelo ARIMA(0, 1, 0) é o passeio aleatório (random walk).
